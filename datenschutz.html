6. Plugins und Tools

Google Fonts (lokales Hosting)

Diese Seite nutzt zur einheitlichen Darstellung von Schriftarten so genannte Google Fonts, die von Google bereitgestellt werden. Die Google Fonts sind lokal installiert. Eine Verbindung zu Servern von Google findet dabei nicht statt.

Weitere Informationen zu Google Fonts finden Sie unter https://developers.google.com/fonts/faq und in der Datenschutzerklärung von Google: https://policies.google.com/privacy?hl=de.

Bootstrap (lokales Hosting)

Diese Seite nutzt das Framework Bootstrap. Bootstrap ist lokal installiert und eine Verbindung zu weiteren Servern findet nicht statt.

Weitere Informationen zu Bootstrap finden Sie unter https://getbootstrap.com/

Canvas-Confetti (lokales Hosting)

Diese Seite nutzt des weiteren auch das Paket Canvas-Confetti. Auch hier wird auf eine lokal installierte Version zurückgegriffen und keine Verbindung zu anderen Servern hergestellt.

Weitere Informationen zu Canvas-Confetti finden Sie unter https://github.com/catdad/canvas-confetti#readme
